# A-Ransomware-Case

## Objective

In the midst of this high-pressure scenario, a ransomware attack unfolds. 😱 The notorious hacking group, LockByte, has encrypted critical hospital files, demanding a hefty ransom for their release!

### Skills Learned

- Learned how to interpret Security Logs to identify evidence of malicious activity
- Learned how to formulate an analytical question and answer it using data
- Learned how to use Azure Data Explorer (ADX) and Kusto Query Language (KQL) to query logs.
- Pivoting Across Data.
- Development of critical thinking and problem-solving skills in cybersecurity.

### Tools Used

- Security Information and Event Management (SIEM) system for log ingestion and analysis.
- Azure Data Explorer (KQL)
- Network analysis tools (such as Wireshark) for capturing and examining network traffic.
- Telemetry generation tools to create realistic network traffic and attack scenarios.
